### Nubo Deploy

- Deploy TypeScript apps ([Bun](https://bun.sh/) and [Deno](https://deno.land/))
- Globally distributed
- Zero cold starts
- Works with Next.js, Vite, Express, Hono, and more

### Nubo SQL

- Globally distributed data
- Strong data consistency
- MySQL compatible
- Works with Prisma
- Built-in compliance (GDPR, CCPA, LGPD, etc.)
